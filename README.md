# 🔒 Secure Notes Vault (Django Project)

مشروع "خزنة الملاحظات الآمنة" هو تطبيق ويب مبني باستخدام Django، يركز على خصوصية المستخدم وتأمين بياناته الشخصية.

## ✨ المميزات (Features)
* **نظام مستخدمين متكامل:** تسجيل حساب جديد وتأمين الدخول.
* **عزل البيانات (Data Isolation):** كل مستخدم له "خزنة" خاصة به، ولا يمكن لأي مستخدم آخر الوصول لملاحظاته حتى لو حاول التلاعب بالروابط.
* **إدارة الملاحظات (CRUD):** إضافة، عرض، وحذف الملاحظات بسهولة.
* **حماية كلمة المرور:** الاعتماد على معايير Django الصارمة لضمان قوة كلمات المرور.

## 🛠️ التقنيات المستخدمة (Tech Stack)
* **Backend:** Python & Django.
* **Database:** SQLite (Default).
* **Frontend:** HTML5 & CSS3.
